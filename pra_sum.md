# 数组练习的总结

## 一、矩阵操作
- 编程特性，先遍历中心数周围的数，再判断是否在矩形之内
- 注意二维数组构成的矩形的对应坐标，一般都为（line + 1, row + 1）
## 二、九宫格操作  
- 对于二维数组，要记住第一个是行，第二个是列
    ```
    int arr[x][y];
    ```  
- 注意switch语句的贯穿问题
    ```
    switch (var):
     {   case 1:
            语句;
            break;
        case 2:
            语句;
            break;
        default:
            语句;
            break;
     }
    ```  
- 对于在循环中含有输入的情况，可以在循环内部写输入语句，然后用条件语句来进行限制
    ```
    char input;
    for (int i = 0; i < 8; i++)
    {
        intput = getchar();
        if (input == '\n') break;
        operate(keyboard, input, &x, &y);
        printf("%d", y * 3 + x + 1);
    }
    ```
- 要善于使用debug来查看代码的问题，关注动态变化，看代码有哪个部分写的不正确。

## 三、字符串排序
- 排序字符串可用strcmp()函数，strcmp()函数如何构造，可以再进行研究[^1]
- 对于字符串排序，与数的排序有不同之处，因为要 **从头到尾一一比较**,所以可以用另一个数组来存储对应字符串是否已经被排序，每次比较之前要从尚未被排序的字符串中找一个为初始值
- 要注意数据的同时性，避免数据在后面使用到时出现bug
- 字符数组实际存储的字符个数与数组大小之间相差1，即'\0'
